PASOS PARA CONVERTIR HTML A ANGULAR:

copiar todos los ASSETS del TEMPLATE  a la carpeta ASSETS que hay en SRC de angular
copiar todos los LINK tags del HEAD del file INDEX del TEMPLATE , al file INDEX de ANGULAR
copiar todos los SCRIPT tags del BODY del file INDEX del TEMPLATE , al file INDEX de ANGULAR
copiar la CLASS del BODY de HTML a ANGULAR

1) HEADER
ng g c  shared/header --skip-tests -m app -d

copiar el HEADER del file HTML al COMPONENT "Header"


2) FOOTER
ng g c shared/footer -m app --skip-tests -d

copiar el FOOTER del file HTML al COMPONENT "Footer"


3) PAGES:
ng g c pages/portfolio  --skip-tests -m app -d
